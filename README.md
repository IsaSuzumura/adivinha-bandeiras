# Adivinha Bandeiras

Jogo simples de adivinhar o nome do país com sua respectiva bandeira, sendo limitado a 10 perguntas que nunca repetem o país. Feito utilizando API de Países "REST COUNTRIES" e com fins acadêmicos.
